## Typical Workflow

Most competitive insight tools require `competitorId` values, which you get from `get_competitor_businesses` first:

```
1. get_competitor_businesses          → get competitorId values
2. get_competitive_insight_scores     → compare category grades
3. get_competitive_insight_themes     → compare top review themes
4. get_competitive_insight_keywords   → compare top keywords
```

## Tips

- Always call `get_competitor_businesses` first — its `competitorId` values are required by all insight tools.
- Pass `competitorId` as both `competitorId` and `compAccountIds` in the `competitors` list parameter.
- Use `sourceType: "review"` (default) for review-based comparisons or `"social"` for social content.
- Pass `timePeriodSelected: "all"` to include all-time data regardless of date range.
