## Tips

- Both tools require `businessNumbers` — always call [`get_child_locations`](/mcp/tools/business/get-child-locations) first to get valid values.
- If the user doesn't specify which locations to include, present the available locations and ask them to choose before calling these tools.
- Use `get_insight_experience_score_benchmark` to answer "how do we compare to the industry?" and `get_insight_experience_location_info` to answer "which of our locations are improving or declining?"
